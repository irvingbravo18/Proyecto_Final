funcionalidad del proyecto
==============================================
     GENERADOR DE CONTRASEÑAS
==============================================

Se muestra un menú de con dos opciones disponible para el usuario  
1.	Generar Contraseña 
2.	Salir
El usuario podrá escoger una sola opción, en el caso que ingrese valores erróneos el sistema le mostrará una aleta notificando las opciones que debe ingresar.

Ingrese la longitud de la contraseña: 10
El usuario podrá escoger una longitud que comprende entre 8 y 20 caracteres para su contraseña, en el caso que el usuario ingrese un valor menor o superior se mostrará un mensaje en pantalla indicando las opciones correctas a seguir.
